

#   vue组件传值方式及区别

  - props $emit 父组件向子组件传递数据通过props，子组件通过$emit通知
    父组件修改数据。
    props的收集：模板编译的时从vNode中将传递给子组件的所有属性,事件提取
                出来放在propsData上，组件初始化的时候会将属性和事件放在
                $options上。
     事件处理：initEvents() 获取所有的事件 listenersupdateComponentListeners() 跟新组件事件，传入 add=>$on 和 remove=>$off 。组件上写的 @事件名 调用的就是 $on 。 发布订阅模式。            


  - $parent $children 获取当前组件的父组件和子组件
   ( 直接操作父子组件的数据不太好)

  - $attrs $listeners 

  - provide inject 父组件提供变量 子组件注入

  - $refs 放到组件上 获取组件的实例 ，放到元素上 获取真实dom

  - eventBus

  - vuex
