

# 为什么需要虚拟dom ( vnode ) ( 虚拟dom有什么作用 )

  - 虚拟dom就是一个对象，用来描述真实dom
    {
      Tag 标签名
      Data 数据 props
      Children：孩子节点
      Elem：真实节点
      Key：唯一标识 用来做dom diff 
      isStatic ：是不是静态节点
      componentOptions 组件属性
      componentInstance 组件实例
    }  等等 比 console.dir(div) 少太多的属性
  
   -  虚拟dom就是用js对象来描述真实的dom节点，是对真实
