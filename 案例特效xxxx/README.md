## 如何把代码下载到本地（Git Clone）

> 这里以Windows举例，Linux、苹果类似。

### 1. 下载git

下载地址：https://git-scm.com/downloads

下载与当前系统匹配的版本，64位系统请下载64位安装包。

### 2. 安装

直接运行安装包即可，正常一路下一步就可以。

### 3. 创建一个文件夹

可以在任何地方创建

### 4. 进入文件夹，在资源管理器中地址栏输入CMD回车

进入控制台模式

### 5. 克隆代码

执行下列命令代码：

``` bash
git clone https://e.coding.net/dishiduo/hcj/hcj.git

```

显示以下信息表示成功：
